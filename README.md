# 🌎 State Website

A multi-page website showcasing cities in South Carolina, including population data and images. This project was created as part of a web development course.

---

## 🚀 Features
- 🌆 Multiple city pages (Charleston, Columbia, Myrtle Beach)
- 🔗 Navigation between pages
- 🖼️ Images and basic styling
- 📊 Displays population information

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
  
---

## 📚 What I Learned
- Structuring multi-page websites
- Linking pages using HTML
- Styling with CSS
- Organizing files and folders
